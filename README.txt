USB BACKUP AGENT v3.0
=====================

Backup otomatis file yang berubah ke USB.

================================================================================
CARA PAKAI:
================================================================================

UNTUK MAC:
----------
1. Double-click "Backup Now.app"
2. Tunggu sampai selesai
3. Done!

UNTUK WINDOWS:
--------------
1. Double-click "Backup Now.bat"
2. Tunggu sampai selesai
3. Done!

================================================================================
FITUR:
================================================================================

- Backup file yang berubah saja (incremental)
- Skip file besar (>50MB)
- Extension yang di-backup:
  * Gambar: jpg, png, gif, webp, heic, raw, dll
  * Video: mp4, mov, avi, mkv, webm, dll
  * Audio: mp3, wav, m4a, flac, dll
  * Dokumen: pdf, doc, docx, xls, xlsx, pptx, txt, csv, dll
  * Code: py, js, html, css, json, md, dll
  * Lainnya: zip, epub, srt

================================================================================
HASIL BACKUP:
================================================================================

File backup tersimpan di:
  USB:/hasil_backup/YYYY-MM-DD_HH-MM-SS/

Struktur folder dipertahankan:
  Documents/Project/file.py -> hasil_backup/YYYY-MM-DD/Documents/Project/file.py

================================================================================
JIKA USB TIDAK TERDETEKSI:
================================================================================

Pastikan:
1. USB dicolokkan
2. Nama USB adalah "Untitled 1" (default)
3. Atau buat file "_usb_backup_marker.txt" di root USB

================================================================================
KONTAK:
================================================================================

Butuh bantuan? Buat issue di GitHub.
